QIG-w : Goodness of outputs relative to input, Outputs as a whole

Node Details
QIG-w-2 : Degree to which Output Answers Question in Input

Definition:
A better system produces outputs that are more complete and accurate answers to questions provided in the input
Elicitation:
Intrinsic, subjective, relative: Which of these outputs is a more complete and accurate answer to the question provided in the input?
Intrinsic, subjective, absolute: To what degree is this output a complete and accurate answer to the question provided in the input?
Attestations:
In the TREC-8 Question Answering shared task Voorhees et al., 1999 ask human assessors whether answers generated by participating systems contain words that answer the input question.
