QIG-w-3 : Quality as Explanation of Input

Definition:
A better system produces outputs that are more complete and accurate explanations of the inputs
Elicitation:
Intrinsic, subjective, relative: Which of these outputs is a more complete and accurate explanation of the input?
Intrinsic, subjective, absolute: To what degree is this output a complete and accurate explanation of the input?
Attestations:
Sarsa et al., 2022 evaluate automatically generated code explanations by answering the question ``Are all parts of the code explained?'' (Yes / No) and computing the proportion of correctly explained lines out of all the generated explanation lines.
Additional notes and information:
Cf. [QEG-w-4] Goodness as Explanation of System Behaviour; Quality as Explanation of Input only takes the input and output into account, assessing how well the latter explains the former.
