QC : Correctness of outputs

Definition:
The degree to which an output is correct.
Suggested question to evaluators in subjective, intrinsic, absolute mode:
[generic] To what degree is this output correct?
Suggested question to evaluators in subjective, intrinsic, relative mode:
[generic] Which of these outputs is more correct?
Additional notes and information:
For correctness criteria, it is possible to give, in the general case, the conditions under which outputs are maximally correct. E.g. for Grammaticality, an output is maximally correct if it contains no grammatical errors, and for Semantic Completeness, an output is correct if it expresses all the content represented in the input. More correct is always preferrable irrespective of context. E.g. (created example): Which of these two texts is more correct?
