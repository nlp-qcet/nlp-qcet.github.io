QEG-w-1 : Naturalness (outputs as a whole)

Definition:
A better system produces texts that are more natural in a given context and/or for a given subset of speakers
Attestations:
In the E2E shared task (Du≈°ek et al., 2018), systems generated short restaurant descriptions from a meaning representation; these were manually evaluated on a quasi-continuous scale in terms of naturalness.
Additional notes and information:
If an output is not natural, then it isn't likely to be encountered in the given scenario. Cf. [QEG-f-2] Appropriateness: if an output is not appropriate then it shouldn't be used in the scenario.
