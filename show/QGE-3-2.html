Node Details
QGE-3-2 : Usability

Definition:
The degree to which users find the system in which the generated outputs are used easy to use.
Suggested question to evaluators in subjective, intrinsic, absolute mode:
[generic] How usable is this system?
Suggested question to evaluators in subjective, intrinsic, relative mode:
[generic] Which of the versions of this system is more usable?
Additional notes and information:
E.g. direct usability ratings for the system, or questions like "do the output texts contribute to the usability of the system?"
