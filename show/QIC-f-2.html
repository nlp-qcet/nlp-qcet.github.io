Node Details
QIC-f-2 : Inclusion of Keywords (given in input)

Definition:
A better system produces texts that lack fewer of the keywords provided in the input
Elicitation:
Intrinsic, subjective, relative: Which of these texts lacks fewer of the keywords provided in the input?
Intrinsic, subjective, absolute: To what degree does this text contain the keywords provided in the input?
Attestations:
Sasazawa et al., 2023 compute the proportion of outputs that contain all the keywords given in the input in several keyword position controlled text generation systems.
