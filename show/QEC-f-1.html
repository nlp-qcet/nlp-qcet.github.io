Node Details
QEC-f-1 : Adherence to Style Guide

Definition:
A better system produces texts with fewer deviations from a given style guide
Attestations:
Zhang et al., 2022 report an automatic code refactoring system which transforms Python code into more python-idiomatic, functionally equivalent code on the basis of a given set of idioms. They assess each refactored output e.g. by manually checking that it conforms with Pythonic idiom, reporting an accuracy of 0.998.
Additional notes and information:
When assessing [QEC-f-1] Adherence to Style Guide, outputs are normally compared with a system-external style guide onn the basis of which style errors can be identified in outputs, and then counted and aggregated.
