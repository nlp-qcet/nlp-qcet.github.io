QGE-3 : Goodness of outputs relative to system use

Node Details
QGE-3-3 : User satisfaction

Definition:
The degree to which users are satisfied with the system in which the generated outputs are used.
Suggested question to evaluators in subjective, intrinsic, absolute mode:
[generic] How satisfied are you with this system?
Suggested question to evaluators in subjective, intrinsic, relative mode:
[generic] Which of the versions of this system are you more satisfied with?
Additional notes and information:
E.g. in a dialogue system "how satisfied were you with the booking you just made?", "how much did you like the system?", counting number of likes output texts get
