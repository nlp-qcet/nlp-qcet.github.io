Node Details
QEG-w-5 : System Usability as Affected by Outputs

Definition:
A better system is more usable by the user, where compared systems differ only in their outputs
Attestations:
Zheng et al., 2015 evaluate five varied NLP systems using a software usability tool called TURF which records interaction streams and user audio; they also evaluate ease of use on a scale from -1 to 2.
Additional notes and information:
In software development, Usability is commonly defined in terms of the effort required to use a system. The scope of the QCET taxonomy is evaluation of NLP systems, so usability evaluation in this context would vary just the NLP components, not e.g. the user interface or animation design.
