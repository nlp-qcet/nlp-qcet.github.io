QEG-w-5.2 : Task Completion Speed

Definition:
A better system produces outputs that result in faster task completion by the user
Attestations:
Qu & Green, 2002 give users flight reservation tasks to evaluate two versions of a dialogue system in terms of task completion time, finding that the system-initiative version allows users to solve tasks more quickly.
Additional notes and information:
This QC is assessed with a system, a user and a task that the user has to complete with the system. The speed with which the user completes the task is measured. Task Completion Speed is independ o overall interaction duration. Cf. [QEF-w-8] Interaction Completion Speed.
